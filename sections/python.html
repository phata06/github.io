Les modules et les packages

En programmation, on a le plus souvent besoin d'utiliser le code écrit par d'autres développeurs pour effectuer des actions ou résoudre des problèmes.

C'est là qu'on va utiliser les modules et les packages. L'utilisation des modules et packages nous fait gagner beaucoup de temps.

module package import from pip

Ceci est un petit rappel des mots clés :

  • module: fichier avec du code Python
  • package: dossier contenant les modules
  • import: utilisé pour inclure des modules et packages externes dans notre code
  • from: importer directement un élément d'un module
  • pip: gestionnaire de packages Python qui permet de télécharger, installer et gérer des packages

Les modules

On crée notre répertoire de travail nommé DevOps. À l'intérieur on crée un fichier operations.py et y mettre le code suivant :

operations.py 
def somme(a, b):
    return a + b

def division(a, b):
    return a / b

def soustraction(a, b):
    return a - b

def multiplication(a, b):
    return a * b

Dans notre répertoire, on crée un fichier main.py et y mettre le code suivant :

main.py 
from operations import somme, soustraction, division, multiplication

print("Le résultat de la somme est ", somme(2, 6))
print("Le résultat de la soustraction est ", soustraction(10, 3))
print("Le résultat de la multiplication est ", multiplication(2, 4))
print("Le résultat de la division est ", division(24, 2))

On a importé les fonctions qui sont dans le module operations.

Les packages

On va créer des packages. On crée un dossier calculateur dans notre répertoire DevOps. On déplace le fichier operations.py dans le dossier calculateur. On crée un fichier __init__.py (qui peut être vide) dans calculateur.

DevOps/
├── main.py
└── calculateur/
├── __init__.py
└── operations.py

Dans main.py, il faut maintenant passer par le package calculateur pour faire l'import comme suit :

main.py (avec package) 
from calculateur.operations import somme, soustraction, division, multiplication

print("Le résultat de la somme est ", somme(2, 6))
print("Le résultat de la soustraction est ", soustraction(10, 3))
print("Le résultat de la multiplication est ", multiplication(2, 4))
print("Le résultat de la division est ", division(24, 2))
